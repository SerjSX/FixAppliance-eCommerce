Multiple areas, same priority allowed:
First: { "serviceAreaID": 6, "priority": 3 }
Second: { "serviceAreaID": 7, "priority": 3 }

{
  "priority": 5
}

No priority:
{
  "serviceAreaID": 1
}

Priority below minimum:
{
  "serviceAreaID": 1,
  "priority": 0
}

Priority above maximum:
{
  "serviceAreaID": 3,
  "priority": 101
}

Decimal priority:
{
  "serviceAreaID": 5,
  "priority": 5.5
}

Priority as string:
{
  "serviceAreaID": 6,
  "priority": "high"
}

{
